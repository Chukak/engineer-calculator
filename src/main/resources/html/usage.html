Usage.

Button descriptions.

0-9 - numbers.

sin, cos, tg, ctg - trigonometric functions.

√ - root.

ln, log - logarithms.

(, ) - mathematical parentheses.

[, ] - parentheses for declare mathematical functions.

∞, π, ℇ - mathematical constants.

← - backspace.

+, -, ×, ÷, mod, ^ - mathematical operators (addiction, subtraction, multiplication, division, remainder, exponentiation).

., , - dot, comma.

AC - clear.

☰ - open menu.

Help - open help.


Menu.

Calculator - select calculator (by default).

Math limits - calculation mathematical limits.

Set EPS - set epsilon to compare mathematical limit above and below.
