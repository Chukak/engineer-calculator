Mathematical limits.

You can calculate some mathematical limits, that not is indeterminacy. If you have indeterminacy in you limits, you can use a mathematical operations to get rid of indeterminacy.

Limits as ∞/∞, 0/0, ∞ - ∞ and some others cannot be calculated by this calculator.

The mathematical limit expression written in the format lim x -> n, ..., where x - name of variable (can be an another character or word), n - limit strives for, can be number or infinity, ... - expression.

You can write the mathematical limit expression or select it from the menu.
