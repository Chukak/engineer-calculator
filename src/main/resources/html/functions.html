Functions.

All math functions should be write in the format funcname[..]. Brackets [ and ] contains function arguments. The first argument - a math expression, the second and further arguments - function parameters. Example: sqrt[8, 3].


Trigonometric functions.

Sin, Cos, Tg, Ctg may have the second argument, implements unit of measure. This is can be radians (rad)or degrees (deg). Example: sin[1, deg] - calculating sine 1 in degrees. Calculating radians by default.


Root.

sqrt may have the second argument, implements root degree. Example: sqrt[8, 3] - the result is 2. Calculating square root by default.


Logarithm.

log - may have the second argument, implements logarithmic base of the number. Example: log[8, 2] - the result is 3. The default logarithmic base of the number is 10.
